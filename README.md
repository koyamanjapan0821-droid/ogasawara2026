# 小笠原2026 PWA

GitHub Pages公開用の旅のしおりアプリです。

## 更新方法

1. ZIPを解凍
2. `index.html`, `style.css`, `app.js`, `manifest.json`, `service-worker.js`, `README.md` をリポジトリ直下へアップロード
3. 既存ファイルを上書きしてコミット
4. GitHub Pages反映を待つ

## 公開URL

https://koyamanjapan0821-droid.github.io/ogasawara2026/

## 主な機能

- Apple風デザイン
- カウントダウン
- 父島の天気
- 日別旅程
- スポット一覧
- Google Maps起動 / ナビ起動
- 旅行中スポット追加
- 訪問済チェック
- 持ち物チェック
- 予約情報
- 写真付き旅行メモ
- PWA / オフライン対応
