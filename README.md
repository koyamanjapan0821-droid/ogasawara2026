# 小笠原2026 PWA

GitHub Pages公開用の旅のしおりアプリです。

## 公開手順
1. GitHubで `ogasawara2026` というPublicリポジトリを作成
2. このZIPの中身をすべてアップロード
3. Settings → Pages → Branch を `main` / `/root` に設定
4. 数分後に公開URLへアクセス

公開URL予定：
https://koyamanjapan0821-droid.github.io/ogasawara2026/

## iPhoneでアプリ風に使う
Safariで公開URLを開く → 共有 → ホーム画面に追加

## 注意
- 天気はオンライン時のみOpen-Meteoから取得します。
- スポット追加、訪問済チェック、持ち物チェック、メモは端末内保存です。
- 端末やブラウザを変えると保存内容は引き継がれません。
